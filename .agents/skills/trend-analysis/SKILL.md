# 步驟 1／7：趨勢研判（Trend）

判定個股趨勢方向，這是進場的第一道關卡。**盤整一律退出觀察**，不做多。

## 資料來源
執行：`python .claude/skills/_shared/fetch_stock.py <股號>`
（若本次對話已抓過同一股票，直接沿用先前 JSON，**勿重複抓取**。）

讀取 `daily.trend`、`daily.swings`、`weekly.trend`、`daily.latest`、`weekly_resistance`（週線壓力價與距離%）。

## 判定規則（朱家泓初階 CH2／道氏理論轉折波）
1. **多頭** = 頭頭高 + 底底高（最近兩個轉折高點墊高、兩個轉折低點墊高）→ 順勢做多。
2. **空頭** = 頭頭低 + 底底低 → 不做多。
3. **盤整** = 高低點未同向 → ❌ 退出觀察，等突破/跌破。
4. **日線 vs 週線**：兩者皆多頭最佳；若週線空頭、日線多頭，視為空頭反彈，列警示（對應做多戒律7）。
5. **週線壓力空間**（看 `weekly_resistance`）：現價上方最近週線轉折高＝壓力價；`distance_pct < 5%` → 遇週線壓力前勿做多（戒律3）；為 null 表示創新高無上方壓力。

## 輸出格式
```
【步驟1 趨勢研判】
狀態：✅多頭 / ⚠️多頭但與週線背離 / ❌空頭或盤整
日線：<多/空/盤>（理由：頭頭高底底高…）
週線：<多/空/盤>；週線壓力：<價位>（距 <x>%）/ 無（創新高）
最近轉折：列出最後 2 高 2 低（日期+價）
結論：<是否進入下一步>
```

- 若日線非多頭 → 明確標示「不符合做多前提，流程可中止」。
- 接著提示使用者可進行 **步驟2 position-analysis**。
