分類規則（每行恰一類，寧嚴勿漏）：
- **(a) 安全**：模板內零 `${}`（純字面量裝在反引號裡）；`${cond ? 'x y' : 'z w'}` 各分支皆為完整 class token 字面量；`${expr}` 其中 expr 可靜態追溯（同檔變數／映射物件／字面量預設值）且**所有可能值都是完整 class 字串字面量、字面量文字本身位於 App.tsx 或 components/** 內**（purge 是文字掃描，完整字面量在被掃描檔案內即可被抽取，位置不限 className 屬性）。
- **(b) 危險**：插值出現在 class token 內部（`text-${x}-500`、`bg-${c}/20`、任何 `${}` 前後緊貼 token 字元）；runtime 任意值（`w-[${n}px]`、`h-[${h}%]`——JIT 建置期同樣抓不到）；`${expr}` 值域無法靜態枚舉（計算字串、來源追不到）。追不出值域＝直接進 (b)。
- **邊界裁定**：ui/* 元件接收 `className` prop 拼進模板（如 `\`base ${className}\``）屬 (a)——呼叫端傳入值本身是呼叫端檔案內的字面量，會被 content glob 掃到；但若某呼叫端傳入動態組出的字串，該「呼叫端行」進 (b)。variant→class 映射物件（Button/Badge 常見寫法）映射值為完整字面量即 (a)。
