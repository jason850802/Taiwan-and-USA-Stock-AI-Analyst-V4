# 步驟 3／7：K線轉折（K-line Signal）

找出多頭的「關鍵進場K線」訊號。這是扣板機的直接依據。

## 資料來源
沿用本次對話 JSON（未抓過才執行 `python .claude/skills/_shared/fetch_stock.py <股號>`）。
讀取 `daily.recent`（最後6根 OHLC+漲跌幅）、`daily.latest`（change_pct、vol_ratio_vs_prev、is_attack_vol、ma5、收盤對前高）。

## 判定規則
**A. 單一K棒**（初階 CH3）：長紅K(實體大、收最高附近)＝強；長黑K＝弱；十字/紡錘＝變盤；下影槌子＝低檔止跌。

**B. 組合K棒**（進階第一章，看最後 2-3 根）：
- 看漲：低檔長黑+長紅(變盤線)、晨星、上升三法、一星二陽、多頭吞噬。
- 看跌：夜星、下降三法、一星二陰、空頭吞噬（出現則 ❌ 不進場）。

**C. 多頭關鍵進場K線（必要訊號，六六大順④／選股SOP第5項）**：
1. 價漲 **> 2%** 的紅K實體棒（看 `change_pct`）。
2. 量增 **> 昨日 1.3 倍**（看 `is_attack_vol`）。
3. 中長紅實體、收盤確認。
4. **突破 5 均（MA5）且收盤過前一日最高點**（用 recent 比對）。

## 輸出格式
```
【步驟3 K線轉折】
狀態：✅出現多頭關鍵K線 / ⚠️訊號不足 / ❌出現看跌組合
單一K棒：<判讀>
組合K棒：<有無晨星/三法/吞噬…>
關鍵進場K線檢核：價漲>2%？量增>1.3倍？突破5均？過昨高？（逐項✓/✗）
結論：
```
