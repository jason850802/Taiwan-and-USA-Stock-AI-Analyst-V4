# 步驟 2／7：當下位置（Position）

確認趨勢為多頭後，判斷股價在波段中的位置——位置決定風險報酬比。

## 資料來源
沿用本次對話的 fetch_stock.py JSON（未抓過才執行 `python .claude/skills/_shared/fetch_stock.py <股號>`）。
讀取 `daily.swings`、`daily.latest`（close、ma 系列）、`daily.candles`、`daily.retrace`（回檔 1/2 法則欄位）、`daily.position_anchors`（高檔量化錨點）。

## 判定規則（六六大順②／朱家泓波段位置）
多頭分段：**打底 → 初升段 → 主升段 → 末升段**；行進狀態：起漲 / 上漲中 / 短線高檔 / 回檔 / 遇壓 / 止跌 / 遇支撐 / 突破。

判讀要點：
- **打底剛突破 / 初升段起漲**：✅ 最佳進場位置。
- **主升段回檔遇支撐（不破前低、靠近 MA10/MA20）**：✅ 符合「回後買上漲」。
- **末升段 / 高檔量化錨點**（看 `daily.position_anchors`，取代模糊描述）：
  - `rise_from_major_low_pct ≥ 100`：自波段大底起漲**達一倍＝高檔**（淘汰法#5、進階連三紅第5點）→ ⚠️ 不宜追多。
  - `ma5_run_rise_pct > 20`：沿 5 日均線累計漲幅**超過 20%**，遇大量長紅後變盤訊號 → ⚠️ 停利警戒（進階連三紅第6點）。
  - `up_streak ≥ 3`：連漲 3 根以上勿追高（戒律2）。
  - 離 MA20 乖離過大亦屬短線高檔 → ⚠️ 追高風險（戒律8）。
- **盤整區間內**：⚠️ 等突破再進（戒律6）。
- **回檔深度 1/2 法則**（初階 CH2，看 `daily.retrace`）：
  - **弱勢回檔**＝回檔幅度 < 前波漲幅 1/2（`ratio < 0.5`）且不破月線（`below_ma20=false`）、未破前低（`broke_prev_low=false`）→ 多頭續漲，「回後買上漲」正宗場景 ✅。
  - **強勢回檔**＝回檔超過 1/2（`ratio > 0.5`）或跌破月線／前低 → 易走頭頭低盤整，需站回月線再談做多 ⚠️。
- 以最近轉折高低點與當前 close 相對位置，加上距離 MA20 的乖離率輔助判斷。

## 輸出格式
```
【步驟2 當下位置】
狀態：✅佳 / ⚠️追高風險 / ❌不宜
波段位置：<打底/初升/主升/末升段>＋<起漲/回檔/高檔…>
與均線關係：close 對 MA10/MA20 乖離概況
結論：<進場位置是否理想>
```
接著提示可進行 **步驟3 kline-signal**。
