# 步驟 5／7：量價關係（Volume-Price）

「量先價行」。進場攻擊一定要有量，且須排除高檔出貨量與量價背離。

## 資料來源
沿用本次對話 JSON（未抓過才執行 `python .claude/skills/_shared/fetch_stock.py <股號>`）。
讀取 `daily.latest`：volume、vol_ma5、vol_ma10、vol_ratio_vs_prev、is_attack_vol、change_pct；搭配 `daily.recent` 看近幾日量價配合。

## 判定規則（初階 CH6／六六大順⑤）
1. **多頭量價**：價漲量增、價跌量縮 → 健康。
2. **攻擊量**：起漲突破日量能 **> 昨日 1.3 倍**（看 `is_attack_vol`）→ ✅ 進場必要條件之一。
3. **量的性質辨識**：
   - 起漲量／換手量（價漲量增、續強）= 好。
   - 高檔或壓力前爆大量、價不漲 = ⚠️ 調節/出貨量（戒律8：連續急漲高檔大量長紅勿追）。
   - 打底量、止跌量（低檔量縮後出量）= 底部訊號。
4. **量價背離**：價創新高但量縮、或價跌量增於低檔 → ⚠️ 列警示。

## 輸出格式
```
【步驟5 量價關係】
狀態：✅攻擊量配合 / ⚠️量能不足或疑似出貨 / ❌量價背離
今量 / 昨量比：<vol_ratio_vs_prev>（攻擊量：是/否）
今量 vs 量5MA：<放大/萎縮>
量性質：<起漲量/換手量/出貨量/止跌量…>
背離：<有/無>
結論：
```
接著提示 **步驟6 indicator-analysis**。
