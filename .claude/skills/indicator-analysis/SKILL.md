# 步驟 6／7：指標（Indicators）

指標是「確認」工具，用來與價量同向佐證，不單獨作為進場依據。

## 資料來源
沿用本次對話 JSON（未抓過才執行 `python .claude/skills/_shared/fetch_stock.py <股號>`）。
讀取 `daily.latest`：k、d、kd_cross、macd_dif、macd_dea、macd_hist、macd_hist_trend；必要時 weekly 同欄位。

## 判定規則（進階第二章 KD/MACD、第三章缺口、第四章底部型態）
1. **KD（參數 5,3,3）**：
   - ✅ 黃金交叉且 K>D 向上多排；**多頭走勢中 50 以下的金叉都是波段起漲點；50 附近的金叉比 20 附近多方力道更強**。
   - **高檔鈍化（KD 進入 80~100 盤整）**：指標失去參考價值，**回歸價量判斷**（價漲＋量增正常仍偏多）；K 值到 80 若短期未向下轉折，股價常急漲——鈍化是強勢股特徵，**不單獨扣分**。
   - 高檔兩次以上死叉 → 大跌機率高；低檔兩次以上金叉 → 大漲機率高。
   - 背離只在 **20~80 之間**才有效：高檔背離（價創高、K值不創高）→ 多單留意停利出場。
2. **MACD**：
   - ✅ 紅柱延長 / 綠柱縮短（hist 由負轉正、或柱狀放大）、DIF 上穿 DEA、零軸之上偏多。
   - ⚠️ 高檔 MACD 與股價背離（價創高、DIF 不創高）→ 警示。
3. **缺口（加分，進階第三章）**：向上跳空突破缺口且未回補 = 強勢；缺口下沿為支撐。
4. **底部型態（加分，進階第四章）**：W底、頭肩底、底部鎖股圖形完成。

## 輸出格式
```
【步驟6 指標】
狀態：✅指標同向偏多 / ⚠️指標分歧或背離 / ❌指標轉弱（不做空）
KD：K=<> D=<> <黃金交叉/多排/高檔鈍化回歸價量>
MACD：DIF=<> DEA=<> 柱=<> <紅柱延長/綠柱縮短/背離>
缺口/型態：<有無向上缺口、底部型態>
結論：
```
