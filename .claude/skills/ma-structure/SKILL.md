# 步驟 4／7：均線架構（MA Structure）

均線架構決定多頭的「地基」是否穩固。**未站上月線一律不做多**（戒律1）。

## 資料來源
沿用本次對話 JSON（未抓過才執行 `python .claude/skills/_shared/fetch_stock.py <股號>`）。
讀取 `daily.latest`：ma5/ma10/ma20/ma60、各 slope、ma_alignment、price_vs_ma20。必要時看 weekly 同欄位。

## 判定規則（初階 CH4／六六大順③）
1. **多排**：至少 **3線多排**（MA5 > MA10 > MA20），更佳為 4 線（含 MA60）多排。
2. **方向**：MA5/MA10/MA20 **方向向上**（slope 上彎）。
3. **股價位置**：收盤 **站上月線 MA20**。← 未站上 = ❌（戒律1、4）。
4. **均線糾結突破**：若先前多條均線糾結、近期帶量向上突破糾結帶，為強力做多訊號（加分）。
5. **扣抵**：MA 方向受扣抵值影響，向上扣抵低值支撐均線續揚（輔助判斷）。

## 輸出格式
```
【步驟4 均線架構】
狀態：✅3線以上多排且站上月線 / ⚠️多排但月線邊緣 / ❌未站上月線或空排
排列：<ma_alignment>（MA5 / MA10 / MA20 / MA60 數值）
方向：MA5/10/20 <上彎/下彎>
股價 vs 月線：<站上/在下>
糾結突破：<有/無>
結論：
```
接著提示 **步驟5 volume-analysis**。
